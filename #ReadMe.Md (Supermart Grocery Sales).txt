# Supermart Grocery Sales Analysis

This project explores the **Supermart Grocery Sales** dataset from Kaggle to uncover key insights into sales performance, customer behavior, and operational trends.

## Tools Used
- **MySQL**: For data cleaning, transformation, and querying.
- **Tableau Public**: For creating interactive dashboards and visualization.

## Objectives
- Identify top-performing product categories and sub-categories.
- Analyze regional sales distribution and profitability.
- Track sales trends over time.
- Detect areas of improvement for stock and discount strategies.

## Key Features
- Cleaned and normalized the dataset using SQL queries.
- Created calculated fields to improve analysis accuracy.
- Developed a dynamic Tableau dashboard with filters and charts such as:
  - Sales by Category
  - Sales Trend Over Time
  - Sales By Region
  - Profit Vs. Sales
  - Top 10 Products By Sales

## Dataset
- Source: https://www.kaggle.com/datasets/mohamedharris/supermart-grocery-sales-retail-analytics-dataset
- Contains fields such as: Order ID, Customer, Category, Sub Category, City, Order Date, Region, Sales, Discount, Profit, State

## How to Use
1. Import the dataset into MySQL.
2. Run SQL scripts to clean and prepare the data.
3. Connect Tableau to your MySQL database or exported CSV file.
4. Recreate or interact with the dashboard.
